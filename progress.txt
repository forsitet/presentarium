## TASK-011: Backend image upload — DONE (2025-11-20)

### What was done:
- Created `backend/internal/handler/upload_handler.go`:
  - POST /api/upload/image: accepts multipart/form-data with field name "image"
  - Enforces 5 MB limit via http.MaxBytesReader + explicit byte count check
  - Detects MIME type from magic bytes (NOT from Content-Type header): JPEG (FF D8 FF), PNG (89 50 4E 47...), WebP (RIFF...WEBP)
